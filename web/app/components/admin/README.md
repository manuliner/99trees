# Admin components

**Purpose:** Organizer backoffice sections for edition setup, tasks, teams, map, and printables.

- **AdminAccordionSection.vue** — collapsible setup-step panel wrapper
- **AdminEditionHeader.vue** — edition title, status badge, create-edition form
- **AdminEditionControl.vue** — go-live checklist and setup-step navigation
- **AdminEditionSettings.vue** — edition name, slug, field count save
- **AdminCrewPassword.vue** — crew login PIN set/save
- **AdminMapUpload.vue** — festival map image file upload
- **AdminPrintPack.vue** — copy join and crew-login URLs to clipboard
- **AdminTasksSection.vue** — task grid, edit/create, import trigger
- **AdminTaskEditModal.vue** — bilingual task editor with slug and payload validation
- **AdminTasksImportModal.vue** — JSON task-pack import with optional overwrite
- **AdminTeamsSection.vue** — teams table, PIN override, open-turn labels
